## 🏢 M&Aディール一覧

### 📌 ディール1：Unilever、食品事業をMcCormickに売却 ＋ Grüns買収

> **重要ポイント：2週間で正反対の2取引を実行**

| 項目 | 内容 |
|------|------|
| 買収企業 | McCormick & Company（スパイス大手） |
| 対象企業 | Unilever 食品事業部門全体（Knorr、Hellmann's、Marmite、Colman's） |
| 取引金額 | **$44.8B（約6.3兆円）** |
| 内訳 | 現金$15.7B + 株式約$29B |
| 取引構造 | **Reverse Morris Trust（税優遇スキーム）**<br>Unileverが食品部門をスピンオフ→即座にMcCormickと合併<br>旧Unilever株主が合併後企業の約2/3を保有 |
| ブリッジ融資 | Citi、Goldman Sachs、Morgan Stanleyが共同アレンジ |
| 統合後売上 | $20B（約2.8兆円） |
| シナジー | $600M（3年目まで）／バックオフィス統合による人員削減含む |
| マージン予想 | 21% → 23〜25%に拡大見込み |
| 市場反応 | McCormick株 -6%（年初来-26%）、Unilever株 -4% |

**📊 バリュエーション分析**

Unileverは低成長・プライベートブランド競合にさらされた調味料ブランドを**約14倍EBITDA**で売却。GLP-1薬（食欲抑制剤）の普及により加工食品セクターの逆風を先読みした戦略的撤退。

#### 🌱 同時取引：Grüns買収（9日後に発表）

| 項目 | 内容 |
|------|------|
| 買収企業 | Unilever |
| 対象企業 | Grüns（栄養グミスタートアップ） |
| 取引金額 | **$1.2B（約1,680億円）** |
| 創業 | 2023年8月（創業32ヶ月で買収） |
| 成長率 | 600% YoY（2025年） |
| ARR | $300M（2025年10月時点） |
| 収益性 | 創業14ヶ月で黒字化達成 |
| バリュエーション | **約4倍PSR**（Series B時$500M評価→1年で2.4倍） |
| 戦略的位置づけ | Beauty & Wellbeingの「Wellness Collective」に編入（Liquid I.V.、Nutrafol等と同カテゴリ） |

> **💡 戦略的インサイト**
>
> UnileverはMultiple Arbitrageを実行：低成長食品を14倍EBITDAで売却し、超高成長サプリを4倍PSRで買収。GLP-1薬普及で「加工食品の被害者を売り、サプリメントの受益者を買う」という2週間での大胆なポートフォリオ再編成。

### 📌 ディール2：American Ocean Minerals + Odyssey Marine Exploration 合併

> **重要ポイント：深海採掘、米国独自許認可ルートで規制アービトラージ**

| 項目 | 内容 |
|------|------|
| 合併当事者 | American Ocean Minerals（AOM）+ Odyssey Marine Exploration（NASDAQ: OMEX） |
| 取引金額 | **$1B（約1,400億円）評価** |
| 取引構造 | Reverse Takeover（逆さ合併）。AOM株主が支配権取得、OMEXがNASDAQ上場維持 |
| ファイナンス | 転換社債$75.6M、PIPE $156M（Citi、Cantor Fitzgerald） |
| 事業内容 | 太平洋海底からポリメタリックノジュール（Ni、Co、Cu、Mn）を採掘 |
| ライセンス範囲 | Cook Islands EEZ + 米国規制CCZ海域、カバー面積500,000km²超、推定埋蔵量14億トン超 |
| 経営陣 | 会長：Tom Albanese（元Rio Tinto CEO）、CEO：Mark Justh（元JPMorgan/Goldman） |
| 市場反応 | OMEX株価 $0.83 → $2.13（+110〜150%）発表当日、その後大半を吐き出す |

**⚖️ 規制アービトラージ戦略**

国際海底機構（ISA）が3年連続でMining Code策定に失敗する一方、米国は独自ルートを構築（2025年4月トランプ大統領令、2026年1月NOAA許認可規則制定）。AOMは「多国間合意が開く前に国内ルートが閉じる」方に賭ける。なお中国が2025年2月にCook Islandsと競合MOUを締結しており、米国議会が警戒している。

### 📌 ディール3：OpenAI、TBPN買収

> **重要ポイント：IPO前ナラティブインフラ構築として初のメディア買収**

| 項目 | 内容 |
|------|------|
| 買収企業 | OpenAI |
| 対象企業 | TBPN（テック系日次ライブストリーム番組） |
| 取引金額 | **非公開（報道では約$150M前後）** |
| 視聴者 | 日次約70,000人 |
| 2025年売上 | 約$5M → 2026年$30M超ペース |
| 買収後方針 | **広告事業を完全停止**（収益化する意図なく$150M投下） |

**🎯 戦略的意図**

これは財務投資ではなく**IPO前ナラティブインフラ**構築。OpenAIは2026年後半〜2027年初に約$1T（約140兆円）評価でのIPOを予定。セカンダリー株は前回ラウンドから10%ディスカウントで取引中。Ben Thompson「OpenAIのTBPN買収は意味不明」、匿名OpenAI投資家「正直理解できない。気が散るし腹立たしい」。

## 🏨 日本不動産・ホテル投資への示唆

### クロスボーダーM&Aと日本市場への波及

- **Unileverのポートフォリオ再編**：GLP-1薬普及を見据えた食品→ウェルネスへのシフトは、日本の消費財・ヘルスケアセクターでも加速の可能性。インバウンド消費に強いウェルネス・サプリメント企業の評価上昇が見込まれる。
- **Reverse Morris Trust活用**：税務効率的なスピンオフ＋合併スキームは、日本のコングロマリットによる事業再編でも参考に。
- **McCormick型統合**：バックオフィス最適化によるシナジー創出は、日本の統合型ホスピタリティ企業でも適用可能。

### セクター別バリュエーション・キャップレートへの影響

- **マルチプルアービトラージ**：成熟ブランド（14倍EBITDA）vs. 高成長スタートアップ（4倍PSR）の評価格差は、日本のホテル資産でも同様。レガシー旅館（低成長）vs. 新興ブティックホテル（高成長）で評価ギャップ拡大の可能性。
- **GLP-1薬と消費トレンド**：ホテル内F&B（飲食）収益への影響を注視すべき。一方、ウェルネス・スパ・フィットネス併設ホテルへの需要増加見込み。
- **米国株市場の史上最高値**：GS/JPM/Citiのトレーディング収益好調は、クロスボーダー投資家の日本不動産取得余力が依然健在であることを示唆。

### ⚠️ リスク・モニタリング事項

- **FRBと政権の対立激化**：パウエル議長解任圧力による金融政策の不確実性増加 → 長期金利ボラティリティ上昇 → キャップレート拡大リスク
- **GLP-1薬の普及加速**：ホテルF&B収益モデルの見直し必要性
- **OpenAI IPO動向**：$1T評価でのIPO実現ならテック企業資金調達環境改善 → 日本インバウンド・DXスタートアップへの波及効果
- **米中覇権競争（深海資源）**：アジア太平洋の地政学リスク上昇は日本のインバウンド需要・クロスボーダー投資に影響
